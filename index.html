Ramble

Ramble is a collection of experiments using force directed input to control an infinite scroller. Pulling down on the scroller applies a upward or downward force on the objects in the viewport.

Github

https://github.com/foldi/Ramble

Compatibility

Ramble experiments are compatible with the latest version of Chrome.

Examples

Object pooling

With object pooling, the scroller is truly infinite.

Vary Color

Varies the color of the objects..

Vary Size

Varies the size of the object.

Vary Speed

Maps size to mass and applies an acceleration to the objects.

Map mouse proximity to acceleration

Uses mouse proximity to derive an acceleration to apply to the objects.
